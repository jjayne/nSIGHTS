Extract Jacobian

What: Extracts Jacobian data from one or multiple simulations of an nSIGHTS Optimizer Results object.

Why: Jacobian data can be viewed in a list page, or converted to a table for plotting.

Used By: Jacobian to Table data object and Jacobian List list object.

Appearance:

Application: nPost

Input Data: nSIGHTS Optimizer Results

Output Data: Jacobian data

Properties:

nSIGHTS Opt Results To Select From The optimizer results from which the Jacobian matrix is to be extracted is selected.

Index Selection Value Source A selection box containing a list of the available simulations allows the user to select the simulation from which to extract the Jacobian matrix.

Mst Selection of the simulation may be slaved to another extraction object. See Masters and Slaves for more information.

Exp Selection of the simulation may be exposed. See Exposed Controls for more information on exposed controls.
