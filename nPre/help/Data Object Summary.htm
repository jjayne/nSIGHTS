Data Object Summary

Due to the large number of data objects, it is not possible to list all data objects in a single selection menu. Consequently, nSIGHTS categorizes data page objects and requires an additional step (category selection) to get the actual object selection menu. The data object category menus in nPre and nPost are as follows:

nPre nPost
