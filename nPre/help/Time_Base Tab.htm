Time-Base Tab

This tab determines the time-based options for defining sequences:

Sequence time entry In Duration mode, the duration of each sequence will need to be entered as input in the Sequences tab table. In Start Time mode, only the start time of each test sequence will need to be entered. For a complex series of sequences, Start Time mode will generally be simpler, as duration times do not need to be calculated for every sequence.
Start time of first sequence The start time of the first sequence needs to be specified for both Duration and Start Time mode. Note that if the time is input before the units are changed, the time will be converted to the new units.
End time of last sequence The end time of the last sequence only needs to be specified for Start Time mode. Note that if the time is input before the units are changed, the time will be converted to the new units.
