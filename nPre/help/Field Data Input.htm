Field Data Input

At this point, existing field data are imported and plotted using the Auto SetupàField Data Plots command.

  1. Select the Auto SetupàField Data Plots.

  2. A dialog will appear:

    3. Toggle off Flow in the Data to Read, and select OK.

  3. Two pages will be created in the Field Data tab of the Plots & Data Processing input window: one data page and one plot page. In the first data page, P_input, select the Read XY Data object, fPDAT.
    • Use the browse button to find the file "PMP.DAT".
    • Select the Apply button. Data Status will indicate that 251 point have been read.

  4. Within the P_Cartesian plot page, select the Pressure XY Data Series object. Auto setup has already selected the field data pressure and the simulated data pressure. Upon selection of the object, its status will be updated to normal (due to the successful reading of data within the fPDAT object in step 3).

  5. The Sequences Sequence Grid Lines object will plot grid lines defining the beginning/ending of all sequences, once sequences are defined.

  6. To view the field data plot, select the 2D plot window (Windowà Fld::P_Cartesian). The window should appear blank, with both axes set at a scale of 0 to 10. To re-scale the axes, select the Reset View button (), or Axes->Data Limits from the 2D pop-up menu (right click within the 2D plot window).

  7. To add axes labels to the plot, select the 2DplotAnno object in the P_Cartesian 2D plot page. In the Format frame, toggle on the Axis Labels checkbox. In the Labels frame, type Time (hours) for the X Axis label, and Pressure (kPa) for the Y Axis label. Select the Apply button. For the Y axis labels to fit within the plot window, the left margin may need to be increased in the 2D-XY object.

  8. To change the title of the plot, select the Title User Labels object in the P_Cartesian 2D plot page. Change the Enter Title text to Field Pressure Data Time Series. Select the Apply button.
