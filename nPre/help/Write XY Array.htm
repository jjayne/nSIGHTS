Write XY Data Array

What: Writes XY array data to a text file.

Why: Allows XY array data to be used or manipulated outside the nSIGHTS application (for example, in another nSIGHTS application).

Appearance:

Application: nPost

Input Data: XY array data

Output Data: text output file containing XY array data compatible with Read XY Data Array, default file extension *.nXYA

Properties:

XY Data Array Data Source Selects the XY array data to be output.
XY data array output file The path and name of the output file is entered in the text box or the Browse button is used to find the file path using the standard Windows open file dialog.
File Format The output file can be written in text or binary format.
The output file is only written once the Apply button is selected.
