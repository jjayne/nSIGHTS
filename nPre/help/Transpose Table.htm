Transpose Table Columns and Rows

What: Switches the column and row data (i.e. output rows = input columns and output columns = input rows).

Why: To manipulate data.

Used By: Any object using table data.

Appearance:

Application: nPost

Input Data: table data

Output Data: table data

Properties:

Input Table Data Selects the table data for which the columns and rows will be transposed.
X Column Processing If skip X column is selected, a specified column is removed from the table before it is transposed.
