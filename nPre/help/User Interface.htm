User Interface

The nPre or nPost user interface consists of a main window, from which the majority of controls are located. Other top-level windows in the interface include plots, the window list menu and object controls. The user interface description in this section focuses on the main window interface.

Additional user-interface components for both nPre and nPost include a menu bar, a tool bar, an object description area (directly below the menu bar), a message line, a status bar and a window list menu window.
