TreeView

Controls how objects are represented in the object tree.

Single line for menu objects If not checked, the text labels for objects in the tree are presented on two lines. Otherwise a single line is used.
Show for single line Options for object identification used in text labels displayed in a single line.
ID Only the object ID is displayed.
Type The object type is shown.
Both Object ID and type are both displayed.
Show unused objects If selected, objects not used by any other object in the configuration file are denoted by a green circle on top of the object icon.
Show connected objects If selected, objects upstream of the selected object are denoted by a magenta diamond on top of the object icon, and objects downstream of the selected object are denoted with a red square on top of the object icon. Upstream objects refer to objects used as input to the selected object, and downstream objects refer to objects which use the selected object as input.
