Calculate Joint Confidence of Table

What: Calculates confidence of points in a table containing fit results by the F-test method or Log-Likelihood method, relative to a best fit case.

Why: To estimate confidence regions.

Used By: Any object using table data.

Appearance:

Input Data: table data

Output Data: table data

Properties:

Table Data To Use Confidence is calculated for each row of data in the selected table.
SSE Data Column The column containing the fit results, sum-squared-error (SSE), is selected.
Best Case Row Index If not automatically determined, the row index of the best fit case is entered.
# of Data Points Number of data points used for SSE calculation (i.e. number of field points in the fit).
# of Parameters Number of parameters optimized.
Options
automatically determine best fit case If selected, the best fit case with the smallest SSE will be selected as the best fit case. Otherwise, the case selected in the Select Best Fit Case list will be used as the best fit case.
F-test method If selected, confidence is calculated using the F-test method.
Log-likelihood method If selected, confidence is calculated using the Log-likelihood method.
