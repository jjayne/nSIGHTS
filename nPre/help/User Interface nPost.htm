nPost User Interface

The nPost user interface consists of a main window, from which the majority of controls are located. Other top-level windows in the interface include plots, the window list menu and object controls. The user interface description in this section focuses on the main window interface.

In nPost, the main window consists of two main components: an object tree, and the object property window. Additional user-interface components include a menu bar, a tool bar, an object description area (directly below the menu bar), a message line, a status bar and a window list menu window.
