Output Files Input

The Output Files options are located on the UI window (nPre control bar). Within this tab, the available output files are specified.

  1. In the Optimization tab, select the checkbox Write optimization output.

  2. In the File name box, use the browse button to specify the output file location. Name the file Tutorial.nOpt.

  3. Select yes for Store residuals and Store Jacobian data drop-down-boxes.
