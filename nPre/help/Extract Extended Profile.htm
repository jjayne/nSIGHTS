Extract Extended Profile from Case

What: Extracts an extended profile from a case (i.e. select the time step to extract the extended profile). The extended profile will have radius and vertical formation height for axes, with a pressure value at each point of the grid.

Why: To examine pressure results as a function of radius and vertical formation height.

Used By: Any object using extended profiles.

Appearance:

Application: nPost

Input Data: Extract Profile Grid/Case

Output Data: extended profile, XY data containing water table height with radius, real value of extracted time

Properties:

nSIGHTS Profile Results To Select From The profile results from which the extended profile is to be extracted is selected.
Index Selection Value Source A selection box containing a list of the available time steps allows the user to select the time step from which to extract the extended profile.
Mst Selection of the time step may be slaved to another extraction object. See Masters and Slaves for more information.
Exp Selection of the time step may be exposed. See Exposed Controls for more information on exposed controls.
