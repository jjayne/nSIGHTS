Validate XY Data as CDF

What: Checks XY data to see if it fulfils the criteria of a CDF, and outputs that CDF.

Why: CDFs may be modified using XY data objects. This object ensures that the XY data is still a CDF, and converts it back to the CDF data type.

Used By: Any object using XY data or CDFs.

Appearance:

Input Data: XY data

Output Data: CDF

Properties:

Input XY Data The XY data set to be evaluated is selected in the drop-down box.
Options
x data is natural log of original The X data (the values) are the log of the original data used to create the CDF.
