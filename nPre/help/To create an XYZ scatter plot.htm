To create an XYZ scatter plot

  1. Create a 3D plot page (PageàNew 3D XY Plot).
    • Change the name of the page to K and n XYZ Scatter. Select Apply.
    • In the X Axis object, change the Increment Label Format to Scientific and 0.0E+00, and select Apply.
    • In the Z Axis object, change the Increment Label Format to Scientific and 0.0E+00, and select Apply.
    • Select the 3D Axes Label object. Check the Axis Labels box, and type Hydraulic Conductivity (m/s) for the X axis, Flow Dimension for the Y axis labels and SSE for the Z axis labels. Select Apply.

  2. Create a new object in the K and n XYZ Scatter plot page: ObjectàNewàData: Table Series.
    • Select Output table SelectOptResults \Perturb Results as the Table Data.
    • Select Formation conductivity as the X Data Column, select Flow Dimension as the Y Data Column, and select SSE as the Z Data Column.
    • Change the Line to Symbol. Change the symbol to Filled C.
    • Select Apply.

  3. View the plot by selecting the K and n XYZ Scatter plot page from the Window menu, or from the Window List window. The axes limits will need to be reset (select Axes->Data Limits in the 2D window pop-up menu). As well, the plot’s view can be adjusted, using the slider controls at the bottom of the window.

  4. XYZ scatter plots can be created for other parameter combinations, by repeating steps (1) to (3). Confidence limits may also be added, if all three parameters are fitting parameters.
