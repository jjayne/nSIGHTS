Select Curve from File

What: Selects a curve from a curve file. A curve file may contain several sets of curve data.

Why: Allows manipulation or plotting of one curve stored within a curve file.

Used By: Any object using curve data.

Appearance:

Application: nPre

Input Data: Read Curve File

Output Data: curve data

Properties:

Curve File To Select From The curve file from which the curve data are extracted is selected.
Index Selection Value Source A selection box containing a list of the available curve data sets allows the user to select the curve data to extract.
Mst Selection of the curve data may be slaved to another object. See Masters and Slaves for more information.
Exp Selection of the curve data may be exposed. See Exposed Controls for more information.
