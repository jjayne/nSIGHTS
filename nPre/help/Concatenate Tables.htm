Concatenate Tables

What: Joins multiple tables into one table. Tables may be joined to the base table by adding columns or rows to the base table. Each table must have the same number of rows, if adding columns, or the same number of columns, if adding rows.

Why: To join mulitple tables.

Used By: Any object using table data.

Appearance:

Application: nPost

Input Data: table data

Output Data: table data

Properties:

Base Table The table data to which tables will be joined.
Concatenate Operation
Columns Tables are concatenated by columns, that is the columns of the selected tables to concatenate are added to the base table. Each selected table must have the same number of rows as the base table.
Rows Tables are concatenated by rows, that is the rows of the selected tables to concatenate are added to the base table. Each selected table must have the same number of columns as the base table.
Options
skip first column when adding When concatenating by columns, the first column of each selected table is not added to the new table.
Tables to Concatenate Selects the tables to add to the base table.
