System Information Objects

System information objects are intended for use with Data Labels plot objects. It provides a mechanism of identifying output graphics for QA and documentation purposes.

There are four system information object types:

Version The current nSIGHTS version identifier (e.g. VersionPre 0.90M).
Version date The release date of the version.
Today The current date.
Time The current time.
