Dual Scale/Transform

What: Performs mathematical operations on both the X and the Y of XY data.

Why:Unit conversions or other data manipulations.

Used By: Any object using XY data.

Appearance:

Input Data: XY data

Output Data: XY data

Properties:

Options
+ve X only If selected, the scale/transform options are only performed on positive X values, and their corresponding Y values.
The remaining scale/transform options are described in Scale/Transform Objects.
