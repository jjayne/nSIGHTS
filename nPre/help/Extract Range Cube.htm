Extract Range Cube

What: Extracts cube data from one or multiple simulations of an nSIGHTS range results object.

Why: To examine the results of a range simulation with three range variables.

Used By: Any object using cube data.

Appearance:

Application: nPost

Input Data: nSIGHTS Range Results

Output Data: cube data

Properties:

nSIGHTS Range Results To Select From The range results from which the range cube is to be extracted is selected.
Index Selection Value Source A selection box containing a list of the available simulations allows the user to select the simulation from which to extract the range cube.
Mst Selection of the simulation may be slaved to another extraction object. See Masters and Slaves for more information.
Exp Selection of the simulation may be exposed. See Exposed Controls for more information.
