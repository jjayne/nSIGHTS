Time Axis

What: Annotation object that displays an X axis with date and/or time labels, in conjunction with the settings in the X axis default object. It is best displayed when the X axis is not displayed (Control in the 2D Plot Annotation default object).

Why: Standard time and/or date axes annotation.

Used By: Nothing (no object output).

Appearance:

Input Data: none

Output Data: none

Properties:

Axes Time Specification Master/Slave control for the Axes Time Mapping and Axes Time Units controls. Master/Slave controls are described in Masters and Slaves.
Axes Time Mapping Specifies the format used to specify the zero date and time.
Arbitrary The zero date and time is specified in the Zero Date and Zero Time controls.
Calendar Year Day 1 is specified as the first of January of the year specified in the Zero Year control.
Excel Days Day 1 is specified as the first of January, 1900.
Axes Time Units For Arbitrary and Calendar Year, specifies the date or time units for the minimum and maximum values specified in the X Axis default object. The time units are days if the Axes Time Mapping is Excel Days.
Zero Date For Arbitrary time mapping, specifies the zero date in month, day, year format.
Zero Time For Arbitrary time mapping, specifies the zero time in hours, minutes, seconds format.
Zero Year For Calendar Year time mapping, pecifies the zero year. The first of January is assumed to be the zero day and month.
Type If Auto is selected, the major and minor increments are determined automatically.
Major Inc. If not Auto, specifies the major increment units.
# per Inc. If not Auto, specifies the number of major increments.
Minor Inc. If not Auto, specifies the minor increment units.
# per Inc. If not Auto, specifies the number of minor increments.
Increment Label Determines the units for the increment labels (Date, Time, Time/Date, Date/Time).
Time Format Determines the format for time labels.
Time/Date Delimiter Determines the delimiter between time and date labels. \n represents a carriage return.
Date Format Determines the format for date labels.
Axes Label Determines the label for the time axes.
