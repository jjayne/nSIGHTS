NL2SOL Tab

The NL2SOL tab provides an option to use non-normalized input parameters (parameters are normalized for all other algorithms), as well as three coefficients used in the algorithm:

Tuner1 If the actual function decrease is less than or equal to Tuner1 * the predicted function decrease, the algorithm will check for false convergence and consider switching models. A value between 0 and 0.5.
Tuner2 If the actual function decrease is greater than Tuner2 * the predicted function decrease, the current step is accepted. A value between 0 and 0.5.
Tuner3 If the actual function decrease is greater than or equal to Tuner3 * the inner product of the step and the gradient, then the trust region radius is increased. A value between 0.001 and 1.
