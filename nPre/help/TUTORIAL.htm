Tutorial

The tutorial in this section provides a guide to the process of developing an nSIGHTS application for well test analysis. There are many ways nSIGHTS may be used to conduct well test analysis, and this tutorial will use the optimization of a constant-rate pumping test (see test description) as an example.

It should be noted that the focus of this tutorial is accessing the many tools available in nSIGHTS, and not the interpretation of well-test results. The tutorial covers the following topics:

  • The set up of a model run and the creation of diagnostic plots (Entering Model Input).
  • The execution of the model (Executing the Model).
  • Evaluating model results, through a variety of different types of plots and text output (Evaluating Model Results).

Field data for the tutorial is required, and is provided as a text file (PMP.DAT).

Check out the test description before starting tutorial.
