Plot Output

nSIGHTS provides two basic types of graphics output: resolution independent postscript, and bitmap files. Still output images are produced using either PostScript or bitmap files, and animations are recorded using bitmap graphics output.

For all types of plot output, the plot window below the button bar and above the reporting area (2D plot) or attitude control area (3D plot) is extracted.
