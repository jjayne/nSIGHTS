Fit Selection Tab

Once field and simulated data have been paired in fit objects within the Fit tab of the Plots & Data Processing input window, the fit specifications to be used as constraints are selected in the Fit Selection tab.

Fits are selected by checking the Optimize? checkbox, and selecting the appropriate fit specification object in the drop-down list. The AutoFill button will automatically select all the created fit objects.

Each line of the Fit Selection tab indicates an independent optimization. Consequently, if more than one line is selected, the simulation will conduct a multiple fit. To optimize several parameters in one optimization (or single fit), use the Composite Fit object in the Fit Specification/Graphics tab to combine several Single Fit, Sequence Fit and/or Composite Fit objects.
