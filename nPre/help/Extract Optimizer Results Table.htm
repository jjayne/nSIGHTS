Extract Optimizer Results Table

What: Extracts a table containing optimized values, case parameters and/or optimization status from one or multiple simulations of an nSIGHTS Optimizer Results object.

Why: To examine optimization results.

Used By: Any object using table data, and the Optimization Results list object.

Appearance:

Application: nPost

Input Data: nSIGHTS Optimizer Results

Output Data: 2 tables, one containing the basic table results, the second containing the initial and final fitting parameter values for each optimization (start/end output).

Properties:

nSIGHTS Opt Results To Select From The optimizer results from which the optimization results are to be extracted is selected.
Table Columns
Optimized values If selected, includes optimized values in the extraction.
Case par If selected, includes the values of case parameters (e.g. suite, range or sampled values for the simulation) in the extraction.
Opt status If selected, includes optimization status in the extraction.
Start/End Table Output The start/end output table contains the initial and final fitting parameter values for each optimization.
None No start/end table is output.
Regular - Segment Table The start/end table is output. There is one column per fitting parameter, and three rows for each simulation (start value in row 1, end value in row 2, and null delimiter in row 3).
Normalized - Segment Table The start/end table is output, with normalized initial and final optimization values. There is one column per fitting parameter, and three rows for each simulation (start value in row 1, end value in row 2, and null delimiter in row 3).
Regular - Segment Table The start/end table is output. There is a Start and End column for each fitting parameter, and a row for each selected simulation.
Normalized - Segment Table The start/end table is output, with normalized initial and final optimization values. There is a Start and End column for each fitting parameter, and a row for each selected simulation.
Index Selection Value Source A selection box containing a list of the available simulations allows the user to select the simulation from which to extract the optimization results. One or multiple selections may be made.
Mst Selection of the simulations may be slaved to another extraction object. See Masters and Slaves for more information.
Exp Selection of the simulations may be exposed. See Exposed Controls for more information.
None All simulations are unselected.
All All simulations are selected.
Run All simulations with the same run identifier as the currently selected simulation are selected.
Case All simulations with the same case identifier as the currently selected simulation are selected.
Fit All simulations with the same fit as the currently selected simulation are selected.
