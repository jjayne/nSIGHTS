Composite Main Menu

What: Controls the general layout and characteristics of composite plots.

Appearance:

Properties:

ObjectID, Plot Pen Set, Background Pen, Plot Dimensions, and Minimum Plot Margins are as described for the 2D XY Main Menu.

Layout Format
Free-Form Individual plots are placed in a layout rectangle specified by an X/Y origin and width within the CompositeLayout object.
Rows/Columns Individual plots are placed in a grid layout, with the number and spacing of the grid rows and columns specified in the CompositeLayout object.
Layout Format
Show layout Layout rectangles are displayed.
Layout pen The pen color used for the layout lines.
