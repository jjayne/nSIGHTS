Appendix B – Plot Object Descriptions

This Appendix is divided into four sections:

  1. Default Plot Objects

  2. Data Display Plot Objects

  3. Annotation Plot Objects

  4. Active Plot Objects

Default plot objects are automatically created upon creation of a plot page, whereas data display, annotation and active plot objects are created using the Objectà New command. Within the selection menu of the Objectà New command, data display object names are prefixed by Data:, annotation object names are prefixed by Anno:, and active objects have prefixes specific to the object.
