Overview

The general steps to creating a model run, executing a model run, and viewing the output are outlined below.

(1) Open nPre. nPre is used to pre-process input data, select model run options and execute the model run.

(2) Complete input requirements through nPre input Windows. For all simulation modes, the following input is required:

  • define general model options and physical configuration à Configuration input window
  • specify simulation output options à Wells and Output input window
  • import field data and develop constraints and diagnostic plots à Field Data tab of the Plots & Data Processing input window
  • define test sequences à Sequence input window
  • enter parameter values à Parameter input window and if applicable, f(x) Points Parameter input window
  • if suite or range parameters are defined, specify related options à Suite/Range input window
  • specify output files to be created during simulation à Output Files input window
  • create plots for viewing during simulation à Runtime tab of the Plots & Data Processing input window

Depending on the simulation mode, further input may be required:
  • if optimization or forward-range mode is selected, pair field data and simulated data for comparison à Fit tab of the Plots & Data Processing input window and the Fit Selection input window
  • if optimization mode is selected, specify optimization options à Optimization input window
  • if sampling mode is selected, specify sampling options à Sampling input window

(3) Execute the model. Select the Minimal or Verbose command from the Run menu. Minimal shows minimal information during the simulation, whereas Verbose provides detailed information at the cost of slightly increased execution time.

(4) Examine diagnostic plots. If required, make necessary adjustments to the input and repeat step (3).

(5) Close nPre.

(6) Open nPost. nPost is a post-processor for simulation results, and contains many statistical tools to examine the results.

(7) Read simulation results into nPost. Analyze and plot the results.

(8) Examine results. If necessary, return to nPre, make necessary changes to the input, and repeat from step (3).
