2D XY Axes/3D XYZ Axes

What: There are two or three default plot objects to define the plot axes, one for the X axis, one for the Y axis, and one for the Z axis (3D plots only), each with identical object property windows.

Appearance:

Properties:

Increment Style For 2D Axes only. Axes increments are represented by grid lines (Grid), tic marks (Tic), or not plotted (None).
Major Style for labelled increments.
Minor Style of unlabelled increments between major increments.
Axis Type Linear or logarithmic axes can be defined. This dialog prompt is also available on the 2D XY Main Menu object. Changes made here will be automatically updated in the 2D XY Main Menu object.
Increment Label Rotation Increment labels can be horizontal or vertical (left or right).
Axes Limits Defines the domain of the plot.
Auto Axes are adjusted to enclose all defined plot objects.
Linear min The minimum co-ordinate value if a linear axis is used.
Linear max The maximum co-ordinate value if a linear axis is used.
Log min The minimum co-ordinate if a log axis is used.
Log max The maximum co-ordinate if a log axis is used.
Major Increment Setup Distance between labelled increments for linear axes.
Auto Increments are set automatically based on data range.
Major increment Value to use if not Auto.
Minor Increment Setup Number of minor increments between each major increment for linear axes.
Auto Increments are set automatically based on major increment size.
Minor per major Value to use if not Auto.

The following dialog prompts are also available on the 2D XY Main Menu object or the 3D XYZ Main Menu object. Changes made here will be automatically updated in the 2D XY Main Menu/3D XYZ Main Menu object.

Increment Label Format Numeric format for labels at each major increment. Numeric format is discussed further in Formatting Real Numbers.
Cursor Reporting Format For 2D Axes only. Numeric format for the X or Y value of the current cursor location in the cursor reporting area. This format is also used for all reporting plot objects that do not have a specific reporting format. Numeric format is discussed further in Formatting Real Numbers.
