P(t) BE/ET Compensation

What: Removes fluctuations in pressure data caused by barometric data and earth tide data. Requires a unit response function.

Why: To remove the influence of barometric and earth tide fluctuations on field data.

Used By: Any object using XY data.

Appearance:

Input Data: XY data and response function (Create BE/ET Response Function)

Output Data: XY data

Properties:

P(t) Data to Compensate Barometric and earth tide fluctuations are removed from the selected input pressure time series data.
Response Function Response function calculated using baseline pressure data in a Create BE/ET Response Function object.
Barometric P(t) Data XY time series containing barometric pressures.
Earth Tide Data XY time series containing earth tide data.
Status Displays the time spacing and memory associated with the response function.
