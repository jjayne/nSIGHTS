nPre Input Window

The nPre input window contains dialog prompts, object trees and corresponding object property windows or tables to define the input data for a simulation. The input window may be organized into tabs, as shown for the Configuration main tab window:

If there are too many tabs to fit within the main window, scroll buttons will appear adjacent to the tabs in the top right hand corner of the input window.

The nPre input windows for each nPre command are described in detail in nPre Input Windows.
