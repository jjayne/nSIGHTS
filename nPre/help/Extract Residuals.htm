Extract Residuals

What: Extracts residuals (XY data) from one or multiple simulations of an nSIGHTS Optimizer Results object.

Why: To examine optimization residuals.

Used By: Any object using residuals, such as Basic Residual or Calculate Residual Diagnostic, as well as any object using XY data.

Appearance:

Application: nPost

Input Data: nSIGHTS Optimizer Results

Output Data: XY data

Properties:

nSIGHTS Opt Results To Select From The optimizer results from which the residuals are to be extracted is selected.

Index Selection Value Source

 A selection box containing a list of the available simulations allows the user to select the simulation from which to extract the residuals.
Mst Selection of the simulation may be slaved to another extraction object. See Masters and Slaves for more information.
Exp Selection of the simulation may be exposed. See Exposed Controls for more information on exposed controls.
