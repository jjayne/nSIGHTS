Table Columns To XY

What: Extracts two specified columns from a table to create XY data.

Why: To convert table data to XY data.

Used By: Any object using XY data.

Appearance:

Input Data: table data

Output Data: XY data

Properties:

Table Data To Use XY data are extracted from columns of the table data selected.
X Data Column The specified table column is used as the X data.
Y Data Column The specified table column is used as the Y data.
Output Description Provides a description of the object, which will be used as the object type in drop-down lists and the label of the object in a Series Legend.
