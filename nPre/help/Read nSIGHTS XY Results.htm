Read nSIGHTS XY Results

What: Reads an nSIGHTS XY simulation results file (default file extension: *.nXYSim), specified in the Output Files nPre input window.

Why: To examine XY simulation results created in nPre within nPost.

Used By: Extract XY from XY Results

Appearance:

Application: nPost

Input Data: XY simulation results file

Output Data: XY results file

Properties:

XY Simulation Results Data File The name of the file (including the file path) containing the XY results is entered in the text box or the Browse button is used to find the file using the standard Windows open file dialog.
File Status Once the XY results file is loaded, the number of runs and cases in the file are displayed.
