XY Data

What: Displays XY data in a list window.

Why: Allows examination of XY data within a separate window.

Used By: Nothing.

Appearance:

Input Data: XY data

Output Data: none

Properties:

XY Data To List Select the XY data to be displayed in the list window.
Data to Write The XYData ID and column headers can be included in the list output.
XY Data Format Determines the numeric format of the data.
Column Width Column width in the list window can be specified.
