Pen Set

What: Defines a set of pens to be used in plotting. Each pen set consists of 24 pens, each of which may be defined to be different colors. Normally, the default Standard Pen Set is all that is required.

Why: Establishes the color of plot objects.

Used By: All plot definitions.

Appearance:

Input Data: no input

Output Data: pen set

Properties:

Each pen is defined by an ID and a color. Selecting a color causes the color’s current settings to be shown on the sliders in the Pen Color Settings frame. Subsequent slider adjustments update the selected color.

Pen Color Method RGB or HSV
Pen Color Settings The RGB or HSV values for the currently selected pen.
