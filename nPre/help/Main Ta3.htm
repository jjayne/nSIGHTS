Main Tab

The Main tab defines the sampling procedure:

Sampling procedure nSIGHTS supports two sampling routines: Latin Hyper-Cube and Monte-Carlo
Number of trials Number of times input parameter distributions are sampled.
Random number seed A user-selected random seed number allows sampling simulations to be reproducible. A random seed number is automatically provided.
User specified variable correlations
yes The correlation between two variables will be forced to the values specified in the Correlations tab.
no Correlations, if not forced to zero, will be calculated during simulation.
Force non-specified correlations to 0.0
yes Non-specified correlations will be forced to zero. If correlations are not user specified, all correlations will be forced to zero.
no Non-specified correlations will not be forced to any value, and will be calculated during simulation.
Sample This button re-samples the parameters, based on the current setting in this tab. Any objects or plots containing sample data will be updated with the new sampled data.
