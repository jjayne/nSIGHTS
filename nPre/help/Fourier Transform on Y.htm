Fourier Transform on Y

What: Conducts a forward or inverse Fourier transform on Y data.

Why: To observe frequency components of a test response.

Used By: Any object using XY data.

Appearance:

Input Data: XY data

Output Data: XY data

Properties:

XY Fourier Transform The XY data set to apply the Fourier transform is selected.
Operation Either a forward or inverse Fourier transform is calculated.
Inverse Options For inverse Fourier transforms only, Y may be scaled by 2/n, and X may be scaled by the entered value.
Output Description Provides a description of the object, which will be used as the object type in drop-down lists, and the label of the object in a Series Legend.
# of FFT points The number of calculated Fourier transform points is displayed.
