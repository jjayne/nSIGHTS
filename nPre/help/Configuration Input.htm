Configuration Input

As most of the defaults apply to the well test, changes only need to be made in the Main tab and the Default Units tab:

  1. Main Tab: Select Optimization as the Simulation Type.
  2. Default Units: Select hour for the Time units, and USgpm for the Volumetric flow rate.

It is also wise to enter a description of the model run in the Test Description tab.

Save the model configuration as Tutorial.nPre, using the standard Windows save button, or FileàSave. Remember to save the configuration file from time to time.
