Single Scale/Transform

What: Performs mathematical operations on either the X or the Y of XY data.

Why: Unit conversions or other data manipulations.

Used By: Any object using XY data.

Appearance:

Input Data: XY data

Output Data: XY data

Properties:

XY Input Data The input XY data set that is scaled and/or transformed is selected.
Data Components to Operate On
X Data If selected, the X data are scaled and/or transformed.
Y Data If selected, the Y data are scaled and/or transformed.
The remaining scale/transform options are described in Scale/Transform Objects.
