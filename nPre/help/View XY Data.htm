View XY Data

What: Allows the user to view XY data.

Why: Used to view XY data created in another object, such as Read XY Data or Dual Scale Transform.

Used By: Any object using XY data.

Appearance:

Input Data: XY data

Output Data: XY data

Properties:

Input XY Data XY data from the object selected are viewed in this property window.
X/YData Format Specifies the numeric format of the X and Y data. Number formatting options are described in Formatting Real Numbers.
Data are viewed in the table once the Apply button has been selected. To modify XY data, use the Enter XY Data object.
