Executing the Model

Before executing the model, save the nPre configuration file.

To execute the model, select the Verbose command from the Run menu. A window will appear, tracking the model as it conducts the simulations.

As there is only one case in this example, there is no progress bar (a progress bar tracks the number of cases completed for runs with multiple cases).

To view the real-time progress of plots developed in the Runtime and Post-Processing Setup input window, select the Minimize button. The run window and the main window will be minimized, leaving only plot windows on the screen.

The model run should take less than a minute. Once it is finished, look at the Seq:F_01_diag plot, which should now show data for simulated pressure and its derivative:

After examining this plot, close nPre.
